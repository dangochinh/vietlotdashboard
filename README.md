# Vietlott Dashboard

## 📌 Về Dự Án
**Vietlott Dashboard** là một ứng dụng phần mềm bao gồm công cụ thu thập dữ liệu (scraper) và giao diện hiển thị (dashboard) theo dõi kết quả xổ số Vietlott. Dự án giúp quá trình tra cứu, thống kê và cập nhật kết quả diễn ra nhanh chóng, thông minh và trực quan nhất.

## 🚀 Tính Năng Chính
- **Dashboard Tổng Quan:** Hiển thị và thống kê kết quả các đợt quay số một cách rõ ràng, hiện đại.
- **Tra Cứu Nhanh Chóng:** Tìm kiếm và lọc dữ liệu qua các kỳ dễ dàng.
- **Tự Động Cập Nhật Dữ Liệu:** Hệ thống tự động (Automated Ingestion Workflow) lấy dữ liệu kết quả mới nhất mà không lo bị trùng lặp.

## 🛠️ Công Nghệ Sử Dụng
- **Frontend / Giao diện:** Next.js (React), Tailwind CSS.
- **Data Scraping:** Python / Scripts thu thập dữ liệu tự động.
- **CI/CD & Automation:** Tích hợp GitHub Actions để cập nhật dữ liệu tự động định kỳ.
